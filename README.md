# Notification Pattern Abstractions

Notification Pattern é um tipo de design de software responsável por notification o client da aplicação sem criar interrupções inesperadas na execução do processo. Ele resolve problemas de baixa performance e baixa fluidez de utilização do programa, tendo em vista que cria notificações por meio de um fluxo de trabalho inteiro da regra de negócios.

Esse repositório é responsável por criar as abstrações necessárias para encapsular esse pattern, de modo que sua utilização seja feita com a abstração e o conhecimento necessário por outras partes da aplicação.
